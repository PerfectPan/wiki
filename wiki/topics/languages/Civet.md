# Civet

## 摘要

Civet 是一门面向 JavaScript / TypeScript 生态的语言层尝试，定位是以更高层、更顺手的语法来书写代码，再编译回 TypeScript 或 JavaScript。

## 关键点

- journal 里的观察虽然简短，但已经点出两个稳定特征：
  - 它不是独立运行时，而是编译到 TypeScript / JavaScript；
  - 它试图通过语法层吸收一些实用表达能力。
- 因此它更像“语言前端层”的实验，而不是要替换整个 JS 生态栈。
