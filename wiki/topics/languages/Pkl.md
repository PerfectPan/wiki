# Pkl

## 摘要

Pkl 是一门面向配置生成的语言，目标是把原本静态、重复、缺少抽象能力的配置文件提升到一个可组合、可导出到多种后端格式的层次。

## 关键点

- 可以直接把它理解成：它不是单纯“又一种配置格式”，而是试图先用一门语言承载抽象，再生成 JSON / YAML / plist 等具体格式。
- 这类语言最核心的价值在于：
  - 配置不再只能靠复制粘贴；
  - 可以引入导入、组合和复用；
  - 同一份配置逻辑可以输出到不同后端。
